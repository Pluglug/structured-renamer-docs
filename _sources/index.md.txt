# Build names, not type them.

<!-- 従来の「検索置換」型リネームとは異なり、意味的な要素に分解した名前パーツを組み替えながら名前を構築します。 -->
<!-- 特にリギングやテクニカルな3Dプロジェクトにおいて、命名規則の維持が容易になります。 -->

Unlike the traditional search‑and‑replace approach, Structured Renamer builds names by rearranging semantically separated name parts.
This makes it easier—especially in rigging or other technical 3D projects—to keep your naming conventions consistent.

## Supported Data Types

Structured Renamer is designed as a versatile tool for managing naming conventions across various data types within Blender.

**Current Support:**

*   **Objects:** All object types in the 3D Viewport.
*   **Bones:** Including Edit Bones and Pose Bones.

**Future Expansion Plans:**

*   Outliner (various data, modifiers, shape keys, etc.)
*   Node Editor
*   Sequence Editor
*   File Browser

**Your Feedback is Appreciated!**

I want to tailor my development roadmap to your needs. If you have specific data types or features you'd like to see prioritized, please share your thoughts on the [Blender Artists](https://blenderartists.org/t/add-on-structured-renamer/1590670).
